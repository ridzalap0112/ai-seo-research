# Research Collection Plan

Repository: `AI-Powered SEO Content Production`  
File target: `research/collection-plan.md`  
Last updated: 2026-06-18

## Collection Rules

- Save YouTube transcripts in `youtube-transcripts/{expert-slug}/`.
- Save LinkedIn post URLs, screenshots, and notes in `linkedin-posts/{expert-slug}/`.
- Save articles, newsletters, PDFs, decks, webinars, and podcast notes in `other/{expert-slug}/`.
- Prioritize materials from 2024-2026.
- For LinkedIn, capture the post URL, publication date, engagement snapshot, core claim, tactical takeaway, and any comments from other practitioners that add implementation detail.
- Prefer implementation examples, case studies, frameworks, experiments, workflows, prompt systems, content briefs, and measurement methods over generic AI predictions.

## Priority Legend

- `10`: Essential source for the repository's core thesis.
- `8-9`: Strong source with repeatable frameworks or implementation evidence.
- `6-7`: Useful supporting source.
- `1-5`: Lower priority unless needed for a specific gap.
